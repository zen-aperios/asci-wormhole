# asci-wormhole

ASCII wormhole pen package with source and minified build.

## Files

- `index.html` demo page
- `style.css` demo styles
- `script.js` demo script
- `asci-wormhole.js` source library script
- `asci-wormhole.min.js` uglified/minified script
- `asci-wormhole.css` source styles
- `asci-wormhole.min.css` min reference css file
- `HOW_TO_WEBFLOW.md` usage guide

## Webflow Reference Links

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/zen-aperios/asci-wormhole@main/asci-wormhole.min.css" />
```
